# Mattress

_"Hide your cash under the mattress!"_

Nostr relay for Cashu tokens of whitelisted pubkeys.

Based on and inspired by the [haven](https://github.com/bitvora/haven) and [sw2](https://github.com/bitvora/sw2) relays.
